# Otus React Cource

## Run Locally

Clone the project

```bash
  git clone git@github.com:Dragonesis/otus-react-cource.git
```

Go to the project directory

```bash
  cd otus-react-cource
```

Install dependencies

```bash
  npm install
```

Run calculator

```bash
  npm run calc
```

Run main project

```bash
  npm run dev
```

Build main project

```bash
  npm run build
```

Run test

```bash
  npm run test
```

Run linter

```bash
  npm run lint:js
  npm run lint:css
```
